**Icon** — the Gravity Jelly glyph set (Lucide-style: 24×24, 2px round stroke, inline SVG, no dependency). Use for all UI icons: pause, settings, rotate, sound, etc.

```jsx
<Icon name="pause" size={24} />
<Icon name="rotate" size={28} color="var(--color-text-invert)" />
```

Names: `pause play settings rotate rotateCw volume mute music vibrate globe info close back home refresh heart star trophy x2 chevron`. Defaults to `currentColor` so it inherits text color.
